## Features

- **Add and drag waypoints** â€” left-click to add, drag to move  
- **Toggle curves** â€” right-click to switch between straight and Bezier  
- **Edit curves** â€” drag control handles to fine-tune paths  
- **Robot preview** â€” set robot size and simulate movement  
- **Play animation** â€” real-time robot path animation  
- **Save/Load JSON** â€” export and import your custom paths

---

## How to Use

1. Click on the field to add waypoints.  
2. Drag waypoints or control handles to adjust the path.  
3. Right-click on segments to toggle curves.  
4. Set robot dimensions using the input fields.  
5. Press **Play** to start the path animation.  
6. Save your path to a JSON file or load a previously saved one.
